# Welcome to WREAM's Pokemon Battler!
## Introduction
This game pits two random pokemon against eachother in a retro pokemon battle.
<img width="500" height="250" alt="image" src="https://github.com/user-attachments/assets/31e035f1-7153-491d-b165-0420e7e52f11" />
We incorporated features like live animation, elemental type multipliers, chance based battle features, a health point + attack point system, and multiple playable pokemon in a tribute to Nintendo's classic pokemon games.


## Instalation and setup
The GitHub repository for this game can be accessed and cloned [here](https://github.com/olincollege/wream-pokemon-battler.git).

Before running the game, run the following in your python terminal:
```
pip install pygame
pip install pygwidgets
```
After pulling from the repository, run the Pygame_game.py python file to start the game!


## Gameplay instructions:

a. Press one of the four move buttons with LMB to select a move when it is your turn, this will expend AP.

b. Watch your HP (red bar) and AP (blue bar), if you run out of either its game over.

c. Consider your attack type when facing an enemy, certain elements will deal bonus damage against eachother.

d. Each move has a different hit rate, so there is a chance your attack wont land.

e. When the game has ended, use the 'x' in the top right corner to exit the game window.
