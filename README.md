# rsmith2-del.github.io
Welcome to WREAM's Pokemon Battler!

This game pits two random pokemon against eachother in a retro pokemon battle.

Pokemon game setup instructions:

run 'pip install pygame' in terminal.
run 'pip install pygwidgets' in terminal.
run the Pygame_game.py python file.
Gameplay instructions:

a. Press one of the four move buttons with LMB to select a move when it is your turn, this will expend AP.

b. Watch your HP (red bar) and AP (blue bar), if you run out of either its game over.

c. Consider your attack type when facing an enemy, certain elements will deal bonus damage against eachother.

d. Each move has a different hit rate, so there is a chance your attack wont land.

e. When the game has ended, use the 'x' in the top right corner to exit the game window.
